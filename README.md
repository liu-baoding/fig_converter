# 图像格式转换工具

一个使用Inkscape实现的图像格式转换工具，支持多种格式之间的相互转换，并提供简单直观的图形用户界面。

## 功能特点

- **多格式支持**：支持SVG、PNG、JPEG、PDF、EPS、PS和EMF格式之间的转换
- **批量处理**：一次性处理多个文件的转换
- **多输出格式**：可同时输出多种不同格式
- **拖拽操作**：支持文件拖拽，简化操作流程
- **自动检测Inkscape**：自动查找Inkscape安装路径，也支持手动指定
- **进度显示**：提供转换进度和状态反馈
- **错误处理**：完善的错误处理和日志记录

## 环境要求

- Python 3.6+
- Inkscape（必须安装）
- tkinter（Python标准库）
- tkinterdnd2（用于文件拖拽功能）

## 安装方法

### 使用可执行文件（推荐）

1. 从[发布页面](链接待添加)下载最新的可执行文件
2. 双击运行，无需安装

### 从源代码运行

1. 克隆或下载本仓库
2. 安装依赖：`pip install tkinterdnd2`
3. 运行程序：`python main.py`

## 使用说明

1. **启动程序**：双击可执行文件或运行Python脚本
2. **选择输出格式**：勾选希望转换成的目标格式（可多选）
3. **添加文件**：
   - 方式1：将文件直接拖拽到程序界面
   - 方式2：点击"添加文件"按钮，从文件浏览器中选择
4. **开始转换**：点击"开始转换"按钮，程序会自动处理所有文件
5. **查看结果**：转换完成后，输出文件将保存在原始文件的相同目录下

## 常见问题

**Q: 程序无法找到Inkscape怎么办？**
A: 请确保Inkscape已正确安装，并且可执行文件在系统PATH中。如果仍然无法自动找到，程序会提示您手动选择Inkscape可执行文件的位置。

**Q: 支持哪些输入文件格式？**
A: 目前支持SVG、PNG、JPEG、PDF、EPS、PS和EMF格式作为输入，但转换效果取决于Inkscape对这些格式的支持程度。

**Q: 为什么需要Inkscape？**
A: 本程序是Inkscape的图形界面封装，实际的格式转换工作由Inkscape完成，因此必须安装Inkscape。

## 日志和错误排查

程序会生成日志文件`fig_converter.log`，记录程序运行过程中的详细信息。如果遇到问题，请查看此日志文件以获取更多信息。
